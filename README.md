I build AI systems that don't stop at the notebook - research, modeling, full-stack engineering, and live deployment. Currently an **AI Product Developer** at **Rooman Technologies**, building [BluLinked](https://crm.rooman.net) - a blue-collar workforce marketplace - end to end: a FastAPI backend, a Groq LLaMA-3.3-70B voice-interview engine over WebSockets (5 Indian languages), a CRM microservice, and a React dashboard.

My research on hybrid NLP + LLM systems for emotional well-being earned an **Indian Patent** in 2025.
